# aritficial_intelligence_machine_learning
These are some of the tasks that I have completed for my AI-ML uni course. 
I have implemented solutions for the famous frog-leap puzzle, n-queen problem, sliding-block-puzzle (IDA*), a tic-tac-toe game (with minimax)  and the knapsack problem (genetic algorithm). Using machine learning algorithms, I have created models that classify data based on their K-nearest neighbours (kNN) and based on probability (NBC).
